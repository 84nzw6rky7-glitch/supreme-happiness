# supreme-happiness

## LIFETAG app

LIFETAG is a lightweight single-page app to capture meaningful life moments and organize them by tags.

### Features

- Add entries with title, tag, and optional note.
- Mark entries complete or active.
- Filter entries by tag.
- Persist entries in browser local storage.

### Run locally

From the repo root:

`python3 -m http.server 4173`

Then open `http://localhost:4173`.
