# supreme-happiness

Product planning repository for a no-judgment QR profile platform focused on
personality-based communication, consent-based interactions, and privacy-first
social discovery.

## Documents

- `docs/product-requirements.md` - complete product requirements document
  (vision, features, policy, and MVP scope).

## Core idea

Each person gets a persistent QR code profile that can be shared on cards,
merchandise, and links. Scanners can view public profile information, but only
registered users can interact (message, comment, follow). Additional paid QR
types support specialized use cases such as romantic profiles, meetups, and
lost-item recovery.
